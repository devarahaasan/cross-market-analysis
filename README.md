# cross-market-analysis

Summary:

Through this project, I applied Financial Analytics to unify disparate data sources into a single SQLite database. Key takeaways include:

Efficiency: Automated data cleaning saved hours of manual processing.

Precision: SQL filtering allowed for granular market trend analysis.

Visualization: Streamlit provided a high-level BI view of market volatility across different sectors.

Conclusion:

This Cross-Market Analysis tool bridges the gap between raw financial data and strategic insights, showcasing the power of Python and SQL in building scalable, real-time Business Intelligence solutions.
